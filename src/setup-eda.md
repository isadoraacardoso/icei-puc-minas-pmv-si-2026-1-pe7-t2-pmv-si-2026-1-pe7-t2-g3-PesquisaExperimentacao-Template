# Setup para executar o código

**Pré-requisito:** Python 3.14 ou superior. Checar:
```bash
python --version
```

**Criar ambiente virtual(opcional)**
```bash
python -m venv .venv

# Linux/macOS
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

**Instalar dependências**
```bash
pip install -r requirements.txt
```

**Executar código**
```bash
python eda-etapa2.py
```
